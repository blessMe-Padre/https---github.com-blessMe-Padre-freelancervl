Фитнес счетчик

Web приложение на React с авторизацией пользователей.

Ссылки на страницы проекта

  • Посмотреть опубликованную версию
  • Посмотреть код на GitHub

Обзор проекта:

  • Авторизация пользоватей и хранение их данных через firebase и Realtime Database.
  • При регистрации нового пользователя ему выдается уникальный идентификатор, проходит запись в базу и его редиректит на главную.
  • Если пользователь уже был зарегистрирован в приложении, он вводит свои данные и его редиректит на главную страницу. Там он может с помощью счетчика ввести количество повторений и отправить их в базу данных на хранение.
