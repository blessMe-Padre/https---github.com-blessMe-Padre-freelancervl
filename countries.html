REST Countries API

REST Countries API with color theme switcher. Двухстраничное (SPA) приложение в экосистеме React.

Сайт сделан в рамках изучения библиотеки React из сервиса Frontend Mentor

Обзор проекта:

  • Вёрстка с нуля в экосистеме React и создание интерактивных React-компонентов. Синтаксис JSX + CSS-in-JS.
  • Выбор темы (темная, светлая).
  • Работа со state и props для демонстрации состояний страниц.
  • Стилизация через библиотеку styled-components.
  • Кастомный селект с библиотекой React Select.
  • Роутинг - react-router-dom 6-ой версии.
  • API rest countries.com
  • Запросы к серверу с библиотекой Axios.
