Оформление заказа

Небольшой проект - страница интернет магазина «Оформление заказа».

Ссылки на страницы проекта

  • Посмотреть опубликованную версию
  • Посмотреть код на GitHub

Обзор проекта:

  • Сверстана одна страница с адаптивностью. Десктопная, планшетная и мобильная версии
  • Реализована вся логика переключателей, кнопок, бургеров, обработка полей, отправка формы .
  • При успешной отправке форме появляется модальное окно
  • Поле адрес и карта связаны, при вводе адреса на карте отображается маркер. Для данного пункта использовались Dadata и yandex-map.
  • При оформлении заказа происходит сбор информации в объект FormData. Собранные данные выводятся в console.log().
